WHAT YOU'LL LEARN

이 강의에서 배우는 것

AI 도구를 활용해 업무 매뉴얼을 만화로 변환하는 전체 파이프라인을 실습합니다

🖥️

Claude Code 마스터

Anthropic의 AI 코딩 CLI 도구인 Claude Code를 활용하여 프롬프트만으로 코드를 생성하고, 복잡한 자동화 파이프라인을 구축하는 방법을 배웁니다.

✨

Gemini API 활용

Google의 멀티모달 AI인 Gemini API를 사용하여 텍스트를 이미지로 변환하고, 만화 스타일의 비주얼을 자동 생성하는 기술을 익힙니다.

📖

업무 매뉴얼 → 만화 변환

복잡한 업무 프로세스 문서를 누구나 쉽게 이해할 수 있는 만화 형태의 가이드로 자동 변환하는 전체 워크플로우를 실습합니다.

⚡

프롬프트 엔지니어링

만화 생성에 최적화된 프롬프트 설계 기법과 멀티턴 대화를 통한 결과물 품질 향상 전략을 학습합니다.

🎸

바이브코딩 실전

코드를 직접 작성하지 않고 AI에게 의도와 분위기(바이브)를 전달하여 원하는 결과물을 만들어내는 바이브코딩의 핵심 원리와 실전 기법을 체험합니다.

🔄

자동화 파이프라인 구축

문서 파싱부터 만화 생성, HTML 패키징까지 전 과정을 하나의 자동화된 파이프라인으로 연결하는 방법을 배웁니다.

🎯

실무 적용 노하우

신입사원 온보딩 가이드, SOP 매뉴얼, 안전 교육 자료 등 실제 기업 환경에서 바로 활용할 수 있는 응용 사례를 다룹니다.

CURRICULUM

커리큘럼

4개 챕터, 실습 중심의 단계별 학습 과정

01

AI 도구 셋업 & 환경 구성

Claude Code와 Gemini API 설치 및 기본 설정

＋
  • Claude Code 소개 — Anthropic의 AI 코딩 CLI 도구 이해하기
  • 환경 설정 — Node.js, Claude Code 설치 및 API 키 발급
  • Gemini API 시작하기 — Google AI Studio 가입 및 API 키 설정
  • 첫 번째 실습 — Claude Code로 간단한 스크립트 만들어보기
  • 프로젝트 구조 설계 — 만화 생성기의 폴더 구조와 모듈 설계
02

업무 문서 파싱 & 시나리오 생성

텍스트 문서를 만화 시나리오로 변환하기

＋
  • 문서 분석 파이프라인 — PDF/텍스트 문서를 구조화된 데이터로 파싱
  • 핵심 단계 추출 — 업무 프로세스를 핵심 단계로 분리하기
  • 캐릭터 설정 — 만화에 등장할 캐릭터와 상황 설계
  • 시나리오 자동 생성 — Claude Code로 만화 시나리오 프롬프트 작성
  • 대화/나레이션 구성 — 각 장면의 대사와 설명 자동 생성
03

Gemini API로 만화 이미지 생성

멀티모달 AI로 만화 패널 자동 생성하기

＋
  • Gemini 이미지 생성 기초 — 텍스트-투-이미지 API 호출 방법
  • 만화 스타일 프롬프트 — 일관된 캐릭터와 화풍을 위한 프롬프트 설계
  • 패널 레이아웃 구성 — 4컷/6컷 만화 레이아웃 자동 배치
  • 말풍선 & 텍스트 오버레이 — 생성된 이미지에 대화 말풍선 합성
  • 품질 최적화 — 결과물 반복 개선을 위한 프롬프트 튜닝 기법
04

완성 & 배포: HTML 만화 가이드 패키징

최종 결과물을 웹 페이지로 패키징하기

＋
  • HTML 템플릿 자동 생성 — Claude Code로 반응형 웹 페이지 구축
  • 인터랙티브 뷰어 — 스크롤/클릭으로 넘기는 만화 뷰어 구현
  • 전체 파이프라인 통합 — 문서 입력 → 만화 생성 → HTML 출력 자동화
  • 실무 활용 사례 — 온보딩 가이드, SOP 매뉴얼, 교육 자료 제작
  • 이메일 · 팀즈 자동 전송 — 완성된 만화 가이드를 이메일 및 Microsoft Teams로 자동 발송하기
  • 확장 아이디어 — 다국어 지원, PDF 출력, 슬랙/노션 연동
TOOLS

활용 도구

강의에서 사용하는 핵심 AI 도구들

🤖

Claude Code

Anthropic의 공식 AI 코딩 CLI. 터미널에서 자연어로 코드를 생성하고, 파일을 수정하며, 복잡한 프로젝트를 자동으로 구축합니다.

✨

Gemini API

Google의 최신 멀티모달 AI. 텍스트를 이미지로 변환하는 네이티브 이미지 생성 기능으로 만화 패널을 자동 생성합니다.

🐍

Python

파이프라인의 핵심 언어. 문서 파싱, API 호출, 이미지 처리, HTML 생성까지 전 과정을 Python으로 구현합니다.

🌐

HTML / CSS

최종 만화 가이드를 아름다운 웹 페이지로 패키징. 반응형 디자인으로 모바일에서도 완벽하게 표시됩니다.

PROCESS

만화 생성 워크플로우

문서 한 장이 만화 가이드가 되기까지

📄 업무 문서 입력

SOP, 매뉴얼, 프로세스 문서 등 텍스트 기반의 업무 가이드를 입력합니다.

🧠 Claude Code가 시나리오 생성

AI가 문서를 분석하여 핵심 단계를 추출하고, 캐릭터와 대사가 포함된 만화 시나리오를 자동 작성합니다.

🎨 Gemini API가 만화 이미지 생성

각 시나리오를 멀티모달 AI가 만화 스타일의 이미지로 렌더링합니다.

✅ HTML 만화 가이드 완성

생성된 이미지와 텍스트를 반응형 웹 페이지로 자동 패키징하여 누구나 볼 수 있는 만화 가이드가 완성됩니다.
